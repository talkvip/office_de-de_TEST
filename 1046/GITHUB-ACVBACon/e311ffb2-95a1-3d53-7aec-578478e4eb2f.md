
# O campo é muito pequeno para aceitar a quantidade de dados que você tentou adicionar. Tente inserir ou colar menos dados. (Erro 3163)

O campo no qual você tentou inserir ou colar dados não é grande o suficiente para manter os dados. Tente inserir ou colar menos dados.
