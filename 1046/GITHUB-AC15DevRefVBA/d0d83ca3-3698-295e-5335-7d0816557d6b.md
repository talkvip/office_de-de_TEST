
# Membros de ComboBox (Access)


Este objeto corresponde a um controle caixa de combinação. O controle caixa de combinação combina as características de uma caixa de texto e de uma caixa de listagem. Utilize uma caixa de combinação quando você desejar ter a opção de digitar um valor ou selecionar um valor de uma lista predefinida.
