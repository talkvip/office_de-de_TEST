
# WorksheetFunction.T_Dist_2T Method (Excel)

Returns the two-tailed Student t-distribution.


## Syntax

 _Ausdruck_. **T_Dist_2T**( ** _Arg1_**, ** _Arg2_** )

 _Ausdruck_ A variable that represents a **[WorksheetFunction](7b1d5639-363d-632c-2cf0-2232562646b6.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|Erforderlich|**Double**|X - The numeric value at which to evaluate the distribution.|
| _Arg2_|Erforderlich|**Double**|Deg_freedom - An integer that indicates the number of degrees of freedom.|

### Return Value

Double


## Remarks




- If any argument is non-numeric, T_DIST_2T returns the #VALUE! error value.
    
- If deg_freedom < 1, T_DIST_2T returns the #NUM! error value.
    
- If x < 0, then T_DIST_2T returns the #NUM! error value.
