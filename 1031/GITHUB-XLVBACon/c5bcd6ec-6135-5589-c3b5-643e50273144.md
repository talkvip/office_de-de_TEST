
# TextFrame2.Ruler Property (Excel)

Returns a  **Ruler2** object that represents the ruler for the specified text. Read-only.


## Syntax

 _Ausdruck_. **Ruler**

 _Ausdruck_ A variable that represents a **TextFrame2** object.
