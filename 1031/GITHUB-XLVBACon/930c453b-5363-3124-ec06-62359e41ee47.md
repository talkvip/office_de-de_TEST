
# ReplaceText-Eigenschaft

 **True**, wenn Text, der einem Eintrag in der AutoKorrektur-Ersetzungsliste entspricht, automatisch ersetzt wird. **Boolean** -Wert mit Lese-/Schreibzugriff.
