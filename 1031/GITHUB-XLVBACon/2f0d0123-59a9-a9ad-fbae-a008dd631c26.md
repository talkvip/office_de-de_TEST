
# ThreeDFormat.IncrementRotationHorizontal Method (Excel)

Changes the rotation of the specified shape horizontally by the specified number of degrees.


## Syntax

 _Ausdruck_. **IncrementRotationHorizontal**( ** _Increment_** )

 _Ausdruck_ A variable that represents a **ThreeDFormat** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Increment_|Erforderlich|**Single**|Specifies how much (in degrees) the rotation of the shape is to be changed horizontally. Can be a value from ?90 through 90. A positive value moves the shape left; a negative value moves it to the right.|
