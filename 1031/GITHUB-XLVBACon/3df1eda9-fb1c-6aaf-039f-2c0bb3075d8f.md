
# PivotFilter.Delete Method (Excel)

Deletes the filter and removes it from the filter collections of the PivotField and the PivotTable.


## Syntax

 _Ausdruck_. **Delete**

 _Ausdruck_ A variable that represents a **PivotFilter** object.
