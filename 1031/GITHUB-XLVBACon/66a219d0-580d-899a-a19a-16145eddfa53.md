
# Error.Value Property (Excel)

Returns a  **Boolean** value that indicates if all the validation criteria are met (that is, if the range contains valid data).


## Syntax

 _Ausdruck_. **Value**

 _Ausdruck_ A variable that represents an **Error** object.
