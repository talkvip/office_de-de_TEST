
# SparklineGroup.Parent Property (Excel)

Returns the parent object for the specified object. Read-only.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ A variable that represents a **[SparklineGroup](cc694d97-a3d3-3473-2e37-0ede67b97680.md)** object.
