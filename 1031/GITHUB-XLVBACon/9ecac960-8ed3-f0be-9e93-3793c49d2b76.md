
# Application.ProtectedViewWindowResize Event (Excel)

Occurs when any  **Protected View** window is resized.


## Syntax

 _Ausdruck_. **ProtectedViewWindowResize**( ** _Pvw_**, )

 _Ausdruck_ A variable that represents an **[Application](19b73597-5cf9-4f56-8227-b5211f657f6f.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Pvw_|Erforderlich|**[ProtectedViewWindow](6a32240c-c90b-c51a-6f8e-c3ff496b9855.md)**|An object that represents the resized  **Protected View** window.|

### Return Value

Nothing


## Remarks

For more information about how to use event procedures with the  **Application** object, see[Using Events with the Application Object](0063feba-47fd-29be-d2d5-8fcf47e70cbc.md).
