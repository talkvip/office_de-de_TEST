
# Workbook.ModelChange Event (Excel)

Occurs after the Excel data model is changed. 


## Syntax

 _Ausdruck_. **ModelChange** _(Changes)_

 _Ausdruck_ A variable that represents a **Workbook** object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Changes_|Erforderlich|MODELCHANGES|[ModelChanges Object (Excel)](fd2388eb-48ab-c238-2ffa-8c3f6d20fe36.md) object which includes changes made to the Excel data model during the last transaction.|
