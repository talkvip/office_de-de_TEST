
# DataFeedConnection.EnableRefresh Property (Excel)

 **True** if the connection can be refreshed by the user. The default value is **True**. **Boolean** Read/Write


## Syntax

 _Ausdruck_. **EnableRefresh**

 _Ausdruck_ A variable that represents a[DataFeedConnection Object (Excel)](2ccb242b-28d5-3baf-78be-aa8f7478f4b6.md) object.


## Property value

 **BOOL**
