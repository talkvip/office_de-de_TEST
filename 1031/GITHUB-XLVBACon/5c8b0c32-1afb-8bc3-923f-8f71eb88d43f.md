
# WorksheetFunction.Power Method (Excel)

Returns the result of a number raised to a power.


## Syntax

 _Ausdruck_. **Power**( ** _Arg1_**, ** _Arg2_** )

 _Ausdruck_ A variable that represents a **WorksheetFunction** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|Erforderlich|**Double**|Number - the base number. It can be any real number.|
| _Arg2_|Erforderlich|**Double**|Power - the exponent to which the base number is raised.|

### Return Value

Double


## Remarks

The "^" operator can be used instead of POWER to indicate to what power the base number is to be raised, such as in 5^2.
