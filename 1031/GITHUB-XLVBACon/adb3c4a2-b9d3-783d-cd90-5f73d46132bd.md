
# DisplayUnitLabel.Height Property (Excel)

Returns the height, in points, of the object. Read-only.


## Syntax

 _Ausdruck_. **Height**

 _Ausdruck_ A variable that represents a **DisplayUnitLabel** object.


### Return Value

Double
