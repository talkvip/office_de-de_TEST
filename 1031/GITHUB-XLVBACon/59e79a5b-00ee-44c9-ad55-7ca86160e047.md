
# Window.Hwnd Property (Excel)

Returns a  **Long** that indicates the window handle of the specified window. Read-only.


## Syntax

 _Ausdruck_. **Hwnd**

 _Ausdruck_ A variable that represents a **Window** object.


## Property value

 **INT32**
