
# SlicerItem.Caption Property (Excel)

Returns the label text for the slicer item. Read-only.


## Syntax

 _Ausdruck_. **Caption**

 _Ausdruck_ A variable that represents a **[SlicerItem](cb93cd82-fc3a-f6b7-ae64-db6312db649d.md)** object.


### Return Value

String
