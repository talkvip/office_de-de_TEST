
# Shadow-Eigenschaft

True, wenn die Schriftart eine schattierte Schriftart ist oder wenn das angegebene Objekt einen Schatten hat. Boolean-Wert mit Lese-/Schreibzugriff.

 _expression_. **Shadow**
