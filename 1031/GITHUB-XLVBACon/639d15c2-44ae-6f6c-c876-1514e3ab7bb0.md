
# WorkbookConnection.Ranges Property (Excel)

Returns the range of object for the specified  **WorkbookConnection** object. Read-only **Ranges**.


## Syntax

 _Ausdruck_. **Ranges**

 _Ausdruck_ A variable that represents a **WorkbookConnection** object.
