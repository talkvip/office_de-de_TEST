
# Sparkline.Location Property (Excel)

Returns or sets the location of a single sparkline. Read/write


## Syntax

 _Ausdruck_. **Location**

 _Ausdruck_ A variable that represents a **[Sparkline](46951c4f-0eaa-9ce6-9703-eb3c632ea9b1.md)** object.
