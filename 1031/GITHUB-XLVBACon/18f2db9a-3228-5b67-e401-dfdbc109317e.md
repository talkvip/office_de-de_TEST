
# DisplayFormat.Parent Property (Excel)

Returns the parent object for the specified object. Read-only.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ A variable that represents a **[DisplayFormat](c70b5d7f-adf1-e539-a32d-12c920af7c7e.md)** object.
