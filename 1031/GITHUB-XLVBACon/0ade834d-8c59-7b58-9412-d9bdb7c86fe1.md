
# OLEObject.Duplicate Method (Excel)

Duplicates the object and returns a reference to the new copy.


## Syntax

 _Ausdruck_. **Duplicate**

 _Ausdruck_ A variable that represents an **OLEObject** object.


### Return Value

Object
