
# CubeField.CurrentPageName Property (Excel)

Returns or sets the page name for a CubeField. Read/write  **String**.


## Syntax

 _Ausdruck_. **CurrentPageName**

 _Ausdruck_ A variable that represents a **CubeField** object.
