
# ListObject.XmlMap Property (Excel)

Returns an  **XmlMap** object that represents the schema map used for the specified table. Read-only.


## Syntax

 _Ausdruck_. **XmlMap**

 _Ausdruck_ A variable that represents a **ListObject** object.
