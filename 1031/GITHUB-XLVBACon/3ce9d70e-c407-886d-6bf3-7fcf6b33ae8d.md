
# WorksheetFunction.Combina Method (Excel)

Returns the number of combinations with repetitions for a given number of items.


## Syntax

 _Ausdruck_. **Combina** _(Arg1,_ _Arg2)_

 _Ausdruck_ A variable that represents a[WorksheetFunction Object (Excel)](7b1d5639-363d-632c-2cf0-2232562646b6.md) object.


### Parameters



|**Name**|**Required/Optional**|**Data type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|Erforderlich|DOUBLE|The total number of items.|
| _Arg2_|Erforderlich|DOUBLE|The number of items in each combination.|

### Return value

 **DOUBLE**
