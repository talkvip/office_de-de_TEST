
# ShapeNodes.Item Method (Excel)

Returns a single object from a collection.


## Syntax

 _Ausdruck_. **Item**( ** _Index_** )

 _Ausdruck_ A variable that represents a **ShapeNodes** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Erforderlich|**Variant**|The name or index number for the object.|

### Return Value

A  **[ShapeNode](c8b60d74-f11f-1659-30a3-6e180eb8bd58.md)** object contained by the collection.
