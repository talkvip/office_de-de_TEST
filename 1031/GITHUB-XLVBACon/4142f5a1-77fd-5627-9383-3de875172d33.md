
# PivotFilter.Value2 Property (Excel)

This property is a user-supplied parameter to define a filter for a PivotField. Read/write  **Variant**.


## Syntax

 _Ausdruck_. **Value2**

 _Ausdruck_ A variable that represents a **PivotFilter** object.
