
# HeaderFooter.Text Property (Excel)

Returns or sets a  **Text** object that represents text included in the specified header or footer. Read/write.


## Syntax

 _Ausdruck_. **Text**

 _Ausdruck_ A variable that represents a **HeaderFooter** object.
