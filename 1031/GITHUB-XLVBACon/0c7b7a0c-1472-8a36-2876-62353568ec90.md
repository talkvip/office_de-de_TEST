
# PivotFilter.FilterType Property (Excel)

Specifies the type of filter to be applied. Read-only  **xlPivotFilterType**.


## Syntax

 _Ausdruck_. **FilterType**

 _Ausdruck_ A variable that represents a **PivotFilter** object.
