
# ChartView.Sheet Property (Excel)

Returns the sheet name for the specified  **[ChartView](2e59e8c1-f1cd-1589-ae36-22d6c5dccbf6.md)** object. Read-only.


## Syntax

 _Ausdruck_. **Sheet**

 _Ausdruck_ A variable that represents a **ChartView** object.
