
# Chart.ProtectContents Property (Excel)

 **True** if the contents of the sheet are protected. For a chart, this protects the entire chart. To turn on content protection, use the **[Protect](5f46d721-021b-d615-12c6-78aab49df500.md)** method with the _Contents_ argument set to **True**. Read-only **Boolean**.


## Syntax

 _Ausdruck_. **ProtectContents**

 _Ausdruck_ A variable that represents a **Chart** object.
