
# ChartCategory.Parent Property (Excel)

Returns the parent object for the specified object. Read-only.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ A variable that represents a **ChartCategory** object.


## Property value

 **OBJECT**
