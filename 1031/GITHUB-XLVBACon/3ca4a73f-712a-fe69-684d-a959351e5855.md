
# Range.FlashFill Method (Excel)

TRUE indicates that the Excel Flash Fill feature has been enabled and active.


## Syntax

 _Ausdruck_. **FlashFill**

 _Ausdruck_ A variable that represents a **Range** object.


### Return value

 **VOID**
