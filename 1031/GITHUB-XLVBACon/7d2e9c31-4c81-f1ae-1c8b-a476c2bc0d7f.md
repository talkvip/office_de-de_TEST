
# Name.Comment Property (Excel)

Returns or sets the comment associated with the name. Read/write  **String**.


## Syntax

 _Ausdruck_. **Comment**

 _Ausdruck_ A variable that represents a **Name** object.


## Remarks

The comment text cannot exceed 255 characters.
