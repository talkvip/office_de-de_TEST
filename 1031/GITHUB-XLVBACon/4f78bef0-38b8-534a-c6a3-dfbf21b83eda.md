
# TextFrame2.Column Property (Excel)

Returns the  **TextColumn2** object that represents the columns within the text frame. Read-only.


## Syntax

 _Ausdruck_. **Column**

 _Ausdruck_ A variable that represents a **TextFrame2** object.
