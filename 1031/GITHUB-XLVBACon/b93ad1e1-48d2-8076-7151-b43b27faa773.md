
# LegendKey.Delete Method (Excel)

Deletes the object.


## Syntax

 _Ausdruck_. **Delete**

 _Ausdruck_ A variable that represents a **LegendKey** object.


### Return Value

Variant


## Remarks

Deleting a  **[LegendKey](2d806a8f-2fed-e6f6-bb76-7339fa692cbb.md)** object deletes the entire series.
