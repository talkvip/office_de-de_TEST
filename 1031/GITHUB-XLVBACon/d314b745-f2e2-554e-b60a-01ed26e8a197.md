
# Slicer.SlicerCacheType Property (Excel)

Returns the type of the slicer cache - slicer or timeline.  **XlSlicerCacheType** Read-only


## Syntax

 _Ausdruck_. **SlicerCacheType**

 _Ausdruck_ A variable that represents a **Slicer** object.


## Property value

 **XLSLICERCACHETYPE**
