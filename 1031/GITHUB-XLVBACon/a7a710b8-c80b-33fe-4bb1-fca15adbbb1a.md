
# PivotLineCells.Parent Property (Excel)

Returns the parent object for the specified  **PivotLineCells** object. Read-only.


## Syntax

 _Ausdruck_. **Parent**

 _Ausdruck_ A variable that represents a **PivotLineCells** object.
