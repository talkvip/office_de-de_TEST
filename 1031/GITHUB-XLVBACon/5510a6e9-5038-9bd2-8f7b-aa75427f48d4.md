
# Chart.Calculate Event (Excel)

Occurs after the chart plots new or changed data, for the  **Chart** object.


## Syntax

 _Ausdruck_. **Calculate**

 _Ausdruck_ A variable that represents a **Chart** object.


### Return Value

nothing
