
# WorksheetFunction.Acos Method (Excel)

Returns the arccosine, or inverse cosine, of a number. The arccosine is the angle whose cosine is  _Arg1_. The returned angle is given in radians in the range 0 (zero) to pi.


## Syntax

 _Ausdruck_. **Acos**( ** _Arg1_** )

 _Ausdruck_ A variable that represents a **WorksheetFunction** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|Erforderlich|**Double**|The cosine of the angle you want and must be from -1 to 1.|

### Return Value

Double


## Remarks

If you want to convert the result from radians to degrees, multiply it by 180/PI() or use the [Degrees](3f66157e-0d2d-f746-1d73-ee7093e6c759.md) method.
