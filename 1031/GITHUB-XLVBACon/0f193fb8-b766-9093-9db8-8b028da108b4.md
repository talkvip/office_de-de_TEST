
# Workbook.DefaultSlicerStyle Property (Excel)

Specifies the style from the  **[TableStyles](952da370-51cb-b1e0-a413-15cb558099b5.md)** object that is used as the default style for slicers. Read/write.


## Syntax

 _Ausdruck_. **DefaultSlicerStyle**

 _Ausdruck_ A variable that represents a **Workbook** object.


### Return Value

Variant
