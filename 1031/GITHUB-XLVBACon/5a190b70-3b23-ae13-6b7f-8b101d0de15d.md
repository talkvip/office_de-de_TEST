
# ColorFormat.RGB Property (Excel)

Returns or sets a  **Long** value that represents the red-green-blue value of the specified color.


## Syntax

 _Ausdruck_. **RGB**

 _Ausdruck_ A variable that represents a **ColorFormat** object.
