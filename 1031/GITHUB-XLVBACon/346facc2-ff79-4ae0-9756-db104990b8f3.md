
# SparklineGroup.Type Property (Excel)

Gets or sets the type of sparkline for the group. Read/write.


## Syntax

 _Ausdruck_. **Type**

 _Ausdruck_ A variable that represents a **[SparklineGroup](cc694d97-a3d3-3473-2e37-0ede67b97680.md)** object.


### Return Value

XlSparkType
