
# TextFrame2.ThreeD Property (Excel)

Returns a  **[ThreeDFormat](9cb41236-6aba-4d6c-a54c-5e177657c8d1.md)** object that contains 3-D-effect formatting properties for the specified text. Read-only.


## Syntax

 _Ausdruck_. **ThreeD**

 _Ausdruck_ A variable that represents a **TextFrame2** object.
