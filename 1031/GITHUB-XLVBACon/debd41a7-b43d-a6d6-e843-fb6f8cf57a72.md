
# HiLoLines.Border Property (Excel)

Returns a  **[Border](bca516bf-7c0f-f9df-078d-dfb522f256f3.md)** object that represents the border of the object.


## Syntax

 _Ausdruck_. **Border**

 _Ausdruck_ A variable that represents a **HiLoLines** object.
