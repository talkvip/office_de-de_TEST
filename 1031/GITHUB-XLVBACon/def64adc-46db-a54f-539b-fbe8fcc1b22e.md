
# Slicer.Style Property (Excel)

Returns or sets the style currently applied to the specified slicer. Read/write.


## Syntax

 _Ausdruck_. **Style**

 _Ausdruck_ A variable that represents a **[Slicer](577be0f6-4eda-0093-8899-097f3c900383.md)** object.


### Return Value

Variant
