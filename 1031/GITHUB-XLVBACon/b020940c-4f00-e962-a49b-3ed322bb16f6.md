
# PivotLine.PivotLineCells Property (Excel)

Returns a collection of  **PivotCell** objects in a PivotLine. Read-only.


## Syntax

 _Ausdruck_. **PivotLineCells**

 _Ausdruck_ A variable that represents a **PivotLine** object.
