
# Application.SmartArtLayouts Property (Excel)

Returns the set of SmartArt layouts that are currently loaded in the application. Read-only


## Syntax

 _Ausdruck_. **SmartArtLayouts**

 _Ausdruck_ A variable that represents an **[Application](19b73597-5cf9-4f56-8227-b5211f657f6f.md)** object.


### Return Value
