
# ChartObject.Cut Method (Excel)

Cuts the object to the Clipboard.


## Syntax

 _Ausdruck_. **Cut**

 _Ausdruck_ A variable that represents a **ChartObject** object.


### Return Value

Variant


## Remarks

Only embedded charts can be cut.
