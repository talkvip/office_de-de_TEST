
# HasLegend-Eigenschaft

 **True**, wenn das Diagramm eine Legende besitzt. **Boolean** -Wert mit Lese-/Schreibzugriff.


## Beispiel

In diesem Beispiel wird die Legende des Diagramms aktiviert. Anschlie√üend wird der Legende die Farbe Blau zugewiesen.


```
With myChart 
 .HasLegend = True 
 .Legend.Font.ColorIndex = 5 
End With
```
