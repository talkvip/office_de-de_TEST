
# DataTable.ShowLegendKey Property (Excel)

 **True** if the data label legend key is visible. Read/write **Boolean**.


## Syntax

 _Ausdruck_. **ShowLegendKey**

 _Ausdruck_ A variable that represents a **DataTable** object.
