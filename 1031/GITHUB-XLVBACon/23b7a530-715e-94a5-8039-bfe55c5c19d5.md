
# ColorStops.Item Method (Excel)

Returns a single object from the represented collection.


## Syntax

 _Ausdruck_. **Item**( ** _Index_** )

 _Ausdruck_ An expression that returns a **ColorStops** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Erforderlich|**Variant**|The name or index number for the object.|

### Return Value

A  **ColorStops** object contained by the collection.
