
# ShapeNodes.Delete Method (Excel)

Deletes the object.


## Syntax

 _Ausdruck_. **Delete**( ** _Index_** )

 _Ausdruck_ A variable that represents a **ShapeNodes** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Erforderlich|**Integer**|The index of the object to delete.|
