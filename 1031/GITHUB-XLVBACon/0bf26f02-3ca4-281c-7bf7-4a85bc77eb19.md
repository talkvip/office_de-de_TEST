
# OLEObject.Visible Property (Excel)

Returns or sets a  **Boolean** value that determines whether the object is visible. Read/write.


## Syntax

 _Ausdruck_. **Visible**

 _Ausdruck_ A variable that represents an **OLEObject** object.
