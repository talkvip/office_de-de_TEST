
# SparklineGroup.LineWeight Property (Excel)

Gets or sets the thickness of the sparklines in the sparkline group. Read/write.


## Syntax

 _Ausdruck_. **LineWeight**

 _Ausdruck_ A variable that represents a **[SparklineGroup](cc694d97-a3d3-3473-2e37-0ede67b97680.md)** object.


### Return Value

Variant
