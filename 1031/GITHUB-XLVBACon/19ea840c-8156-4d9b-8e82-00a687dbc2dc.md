
# Worksheet.BeforeDelete Event (Excel)

Occurs before the worksheet is deleted.


## Syntax

 _Ausdruck_. **BeforeDelete**

 _Ausdruck_ A variable that represents a **Worksheet** object.


### Return Value

None
