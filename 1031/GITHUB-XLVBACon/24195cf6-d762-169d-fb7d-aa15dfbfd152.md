
# WorksheetFunction.Asin Method (Excel)

Returns the arcsine, or inverse sine, of a number. The arcsine is the angle whose sine is  _Arg1_. The returned angle is given in radians in the range -pi/2 to pi/2.


## Syntax

 _Ausdruck_. **Asin**( ** _Arg1_** )

 _Ausdruck_ A variable that represents a **WorksheetFunction** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|Erforderlich|**Double**|The sine of the angle you want and must be from -1 to 1.|

### Return Value

Double


## Remarks

To express the arcsine in degrees, multiply the result by 180/PI( ) or use the [Degrees](3f66157e-0d2d-f746-1d73-ee7093e6c759.md) method.
