
# PivotFormula.Value Property (Excel)

Returns or sets a  **String** value that represents the name of the specified formula in the PivotTable formula.


## Syntax

 _Ausdruck_. **Value**

 _Ausdruck_ A variable that represents a **PivotFormula** object.
