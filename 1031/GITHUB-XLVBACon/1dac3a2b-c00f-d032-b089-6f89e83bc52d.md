
# ProtectedViewWindow.Activate Method (Excel)

Brings the  **Protected View** window to the front of the z-order.


## Syntax

 _Ausdruck_. **Activate**

 _Ausdruck_ A variable that represents a **[ProtectedViewWindow](6a32240c-c90b-c51a-6f8e-c3ff496b9855.md)** object.


### Return Value

Nothing
