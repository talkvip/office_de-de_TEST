
# WorksheetFunction.Lcm Method (Excel)

Returns the least common multiple of integers. The least common multiple is the smallest positive integer that is a multiple of all integer arguments number1, number2, and so on. Use LCM to add fractions with different denominators.


## Syntax

 _Ausdruck_. **Lcm**( ** _Arg1_**, ** _Arg2_**, ** _Arg3_**, ** _Arg4_**, ** _Arg5_**, ** _Arg6_**, ** _Arg7_**, ** _Arg8_**, ** _Arg9_**, ** _Arg10_**, ** _Arg11_**, ** _Arg12_**, ** _Arg13_**, ** _Arg14_**, ** _Arg15_**, ** _Arg16_**, ** _Arg17_**, ** _Arg18_**, ** _Arg19_**, ** _Arg20_**, ** _Arg21_**, ** _Arg22_**, ** _Arg23_**, ** _Arg24_**, ** _Arg25_**, ** _Arg26_**, ** _Arg27_**, ** _Arg28_**, ** _Arg29_**, ** _Arg30_** )

 _Ausdruck_ A variable that represents a **WorksheetFunction** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|Erforderlich|**Variant**|Number1, number2,... - 1 to 29 values for which you want the least common multiple. If value is not an integer, it is truncated.|

### Return Value

Double


## Remarks




- If any argument is nonnumeric, LCM returns the #VALUE! error value.
    
- If any argument is less than zero, LCM returns the #NUM! error value.
