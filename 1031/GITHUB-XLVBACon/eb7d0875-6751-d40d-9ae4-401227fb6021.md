
# Point.MarkerSize Property (Excel)

Returns or sets the data-marker size, in points. Can be a value from 2 through 72. Read/write  **Long**.


## Syntax

 _Ausdruck_. **MarkerSize**

 _Ausdruck_ A variable that represents a **Point** object.
