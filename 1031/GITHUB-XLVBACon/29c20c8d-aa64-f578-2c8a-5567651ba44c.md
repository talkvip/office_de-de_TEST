
# Worksheet.ListObjects Property (Excel)

Returns a collection of  **[ListObject](46de6c4f-8ce0-0c7d-da59-6e52f5eab612.md)** objects in the worksheet. Read-only **ListObjects** collection.


## Syntax

 _Ausdruck_. **ListObjects**

 _Ausdruck_ A variable that represents a **Worksheet** object.
