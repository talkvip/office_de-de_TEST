
# TickLabels.Format Property (Excel)

Returns the  **[ChartFormat](edac71b7-ed38-6658-2cbf-6493dc1ad3ed.md)** object. Read-only.


## Syntax

 _Ausdruck_. **Format**

 _Ausdruck_ A variable that represents a **TickLabels** object.
