
# WorksheetFunction.Quartile_Exc Method (Excel)

Returns the quartile of the data set, based on percentile values from 0..1, exclusive.


## Syntax

 _Ausdruck_. **Quartile_Exc**( ** _Arg1_**, ** _Arg2_** )

 _Ausdruck_ A variable that represents a **[WorksheetFunction](7b1d5639-363d-632c-2cf0-2232562646b6.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Arg1_|Erforderlich|**Variant**|Array - The array or cell range of numeric values for which you want the quartile value.|
| _Arg2_|Erforderlich|**Double**|Quart - The value to return.|

### Return Value

Double


## Remarks




- If array is empty, QUARTILE_EXC returns the #NUM! error value.
    
- If quart is not an integer, it is truncated. 
    
- If quart ? 0 or if quart ? 4, QUARTILE_EXC returns the #NUM! error value.
    
- MIN, MEDIAN, and MAX return the same value as QUARTILE_EXC when quart is equal to 0 (zero), 2, and 4, respectively.
