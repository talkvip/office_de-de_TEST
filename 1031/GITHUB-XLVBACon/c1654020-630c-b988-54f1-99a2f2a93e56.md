
# Shapes.AddFormControl Method (Excel)

Creates a Microsoft Excel control. Returns a  **[Shape](8f01fcd1-b7d9-5216-2de5-40fb6648a403.md)** object that represents the new control.


## Syntax

 _Ausdruck_. **AddFormControl**( ** _Type_**, ** _Left_**, ** _Top_**, ** _Width_**, ** _Height_** )

 _Ausdruck_ A variable that represents a **Shapes** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Type_|Erforderlich|**[XlFormControl](fad54f9d-4ef2-b2ac-d187-131e5bdd18e1.md)**|The Microsoft Excel control type. You cannot create an edit box on a worksheet.|
| _Left_|Erforderlich|**Long**|The initial coordinates of the new object (in points) relative to the upper-left corner of cell A1 on a worksheet or to the upper-left corner of a chart.|
| _Top_|Erforderlich|**Long**|The initial coordinates of the new object (in points) relative to the upper-left corner of cell A1 on a worksheet or to the upper-left corner of a chart.|
| _Width_|Erforderlich|**Long**|The initial size of the new object, in points.|
| _Height_|Erforderlich|**Long**|The initial size of the new object, in points.|

### Return Value

Shape


## Remarks

Use the  **[AddOLEObject](6e73970f-3c2d-0e4d-8974-14e478bf489a.md)** method or the **[Add](2acd369f-6dd6-0e0e-043c-a691796659a9.md)** method of the **[OLEObjects](e3fcf4bd-7c96-ecb3-dc04-551f7f7348f9.md)** collection to create an ActiveX control.


## Example

This example adds a list box to worksheet one and sets the fill range for the list box.


```
With Worksheets(1) 
 Set lb = .Shapes.AddFormControl(xlListBox, 100, 10, 100, 100) 
 lb.ControlFormat.ListFillRange = "A1:A10" 
End With
```
