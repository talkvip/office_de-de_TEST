
# PictureFormat.Crop Property (Excel)

Returns an  **[Crop](http://msdn.microsoft.com/library/21ac150e-0a8f-c77b-717f-bf38fbced5a3%28Office.15%29.aspx)** object that represents the cropping settings for the specified **[PictureFormat](7e8ec723-b6e0-fdc9-ff4e-22cbb31be4df.md)** object. Read-only


## Syntax

 _Ausdruck_. **Crop**

 _Ausdruck_ A variable that represents a **PictureFormat** object.
